## 使用

``` 
import 'package:blur_glass/blur_glass.dart';
```

```
BlurGlass(
                        marginValue: 5.0,
                        paddingValue: 5.0,
                        child: Row(
                          children: [
                            Icon(Icons.star),
                            Icon(Icons.list),
                            Icon(Icons.zoom_in),
                          ],
                        )
                    )
```
